<h1 align="center">Smart Code - POC</h1>

Uma POC simples para testarmos a performance da leitura e alteração de QRCode

## Iniciando :zap:
    git clone https://gitlab.com/techlab1/konsulta-web.git
    cd konsumo-web
    
## Atualizando dependências :zap:
    yarn install
    npm install
    
## Iniciando o projeto 
    npm run dev
